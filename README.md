# Mobile Test Automation Framework (Appium)

## Overview
This project is a mobile test automation framework for Android applications using Appium, Java, TestNG, and Maven.

## Tech Stack
- Java
- Appium
- TestNG
- Maven
- Android Emulator

## Framework Design
This project uses the Page Object Model (POM) design pattern:
- Page classes contain locators and actions
- Test classes contain test logic

## Test Scenario
1. Open the app
2. Enter name
3. Tap button
4. Verify greeting message

### 1. Start emulator
